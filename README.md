# Java Member System

## 📌 專案介紹

使用 Java 21 與 Spring Boot 開發的會員管理系統。

此專案作為 Java 後端學習與實戰練習專案，逐步實作會員系統常見功能，並學習 Spring Framework、Spring Boot、JPA、Redis、Docker、CI/CD、AWS、GCP 與 Kubernetes 等技術。

---

## 🚀 技術

* Java 21
* Spring Boot 3
* Spring MVC
* Spring Data JPA
* Hibernate
* MariaDB
* Maven

---

## ✨ 功能

* 會員 CRUD
* 資料驗證（Validation）
* 全域例外處理（Exception Handling）
* JPA 資料庫操作
* RESTful API

---

## 🔜 預計開發功能

* JWT 登入驗證
* Spring Security
* Redis 快取
* 會員點數系統
* 會員等級升級機制
* 排程任務（Scheduler）
* Docker 容器化
* GitHub Actions CI/CD
* AWS 部署
* GCP 部署
* Kubernetes
